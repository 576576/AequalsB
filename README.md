# A=B  Java interpreter 

该项目为 esolang`A=B`的`JAVA`解释器实现.

A=B是一门只有一条指令的编程语言：A=B，意思是将A替换为B。

本项目实现的A=B指令集包括：
- `A=B` 将A替换为B
- `once start end return` 拓展关键字

-----
## 已知缺陷
- 代码及输入合法性未经检查, 输入特殊字符(*+等)时报错.  
*不输入特殊字符即可*

本项目支持预编写输入输出调试。  
这是一个练习项目，不涉及商业使用，侵删。
